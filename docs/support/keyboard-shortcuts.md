# Keyboard shortcuts

Speed up your workflow in Plane with keyboard shortcuts. Access this list anytime by pressing `Cmd + /` (Mac) or `Ctrl + /` (Windows/Linux).

## Work item actions

| Action | Shortcut |
|--------|----------|
| Change state | `S` |
| Change priority | `P` |
| Assign to | `A` |
| Assign to me | `I` |
| Change estimate | `⌘ + E` |
| Add to cycle | `⌘ + C` |
| Add to modules | `⌘ + M` |
| Add labels | `L` |
| Subscribe to notifications | `⌘ + S` |
| Delete | `⌘ + ⌫` |
| Copy ID | `⌘ + .` |
| Copy title | `⌘ + ⇧ + '` |
| Copy URL | `⌘ + ⇧ + ,` |
